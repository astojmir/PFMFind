Examining search hitsÂ¶

For each fragment-based query, the retrieved database fragments can be examined through the Hits tab tab. It consists of the sequence viewer on the top, navigation pannels on the left and the text area on the right that shows the hits in BLAST-like format.

Hits tab

The residues in the sequence viewers are highlighted according to the availability of search results.

Color Interpretation
Results are available for the currently selected fragment length and iteration
Results are available for a later iteration than selected, for the currently selected fragment length
Results are available for an earlier iteration than selected, for the currently selected fragment length
Results are not available for the currently selected fragment length, but are available for another length
